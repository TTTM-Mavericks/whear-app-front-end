## System requirements
- Node.js v18
- pnpm (npm/yarn) v10

## Setting

#### Clone repository
`git clone https://github.com/TTTM-Mavericks/whear-app-front-end.git`

#### Install node_modules
`npm install`
#### Open on localhost:3000
`npm run dev`
